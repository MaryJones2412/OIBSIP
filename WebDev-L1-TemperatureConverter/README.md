# Temperature Converter Website

## Description
A professional Temperature Converter Website built using HTML5, CSS3, and JavaScript. It converts temperature values between Celsius, Fahrenheit, and Kelvin with real-time validation and a responsive user interface.

## Features
- Convert Celsius, Fahrenheit, and Kelvin
- Input validation
- Absolute zero validation
- Convert and Reset buttons
- Responsive design
- Glassmorphism UI
- Animated gradient background

## Technologies Used
- HTML5
- CSS3
- JavaScript
